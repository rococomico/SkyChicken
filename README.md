## 操作方法
キーボードを用いて操作します。操作方法は以下の通りです。
* 左へ移動: `Left` / `A`
* 右へ移動: `Right` / `D`
* ジャンプ: `Space` / `Z`
* メニューを開く: `Escape` / `X`
* 選択肢の変更: `Up` / `Down`

また、スマートフォン・タブレットでも遊ぶことができます。ただし、キーボードと比較して操作がしづらくなります。
* 左へ移動: `Left`
* 右へ移動: `Right`
* ジャンプ: `A` / `B`
* メニューを開く: `X` / `Y`
* 選択肢の変更: `Up` / `Down`
